Domain Names

Definition:

Domain Name acts as an identification network domain. It is used represent server computer hosting a website. It is easily recognizable and memorable names. Domain name is address in string format.If you type a domain name in your search engine it will look for that domain name in local dns if not found in local dns then it will ask other dns if you know this domain name. If second dns knows it will tell the local dns the address and store it in cache for the next time.After the local gets address of domain name it routes the user to destination.

There are different types of domain names:

  • Top Level Domain
  • Country Code Top Level Domain
  • Sponsored Top Level Domain
Example:

Suppose you use www.google.com, www.facebook.com, www.netflix.com,etc. if there were no domain names you would have to remember the IP address for each network which is difficult to. Hear i have mentioned only some there are people who use dozens of different sites so it would be impossible to remember all.

Uses:
  • It is easier to remember the networks instead of numerical format.
  • Domain names can be identify individual or organization.
  • We are able to uniquely identify the networks.
