IP Addresses

Definition:

All the networks in internet have some addresses. An address on the internet is just a number that's unique to each device on the network. It's similar to the mail system as you need to know the receivers address to send message. The most important protocol is Internet protocol which is used for IP address. Network administrators assign IP address to each device. There are different types of IP versions as it is evolving like IPv4, IPV6,etc.

Example:

Suppose you want to sent data from your computer to your friends computer. For that you will need to know the IP address of your friend. Once, you know that while sending data IP packet carry the IP address of both sender and receiver. As if your friend wants to reply back he/she will have your address.

Uses:
  • It helps identify the host.
  • Location of the host in the network.
