W3C

Definition:

The World Wide Web Consortium (W3C) is an international community where Member organizations, a full-time staff, and the public work together to develop Web standards. Led by Web inventor and Director Tim Berners-Lee and CEO Jeffrey Jaffe, W3C's mission is to lead the Web to its full potential.

Example:

We ourselves are watching the web grow. At first the web only had text then pictures, graphics, etc. came in. It is evolving rapidly as the W3C are working on it.

Uses:
  • Focuses on improvement of Web.
  • Web enables human communication, commerce, and opportunities to share knowledge.
  • Benefits should be available to all people.
