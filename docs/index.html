What is Internet?

Internet can be described as network of networks. Internet basically is a connection between machines using software and hardware devices. Internet uses binary code as method of communication. All the information that is there on internet is made of Bits (i.e. 0 and 1). To transmit these bits with require bandwidth. There are three types of propagation broadband, wireless and radio.


How does Internet work?

First, we take the data that that is to be sent over the network and divide into many small parts called as packets. Packet contains information about the source and destination address. Packets reach the destination from different paths and are reassembled at the destination. Receiver sends acknowledgement to sender.
